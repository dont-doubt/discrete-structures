# Задание 1

Пользователь вводит число n, программа выводит булеву функцию от n аргументов

На странице должны быть:

-	Поле для ввода
-	Поле вывода
-	Поле с описанием задания «Введите количество аргументов, чтобы получить булеву функцию»
-	Справка (можно сделать в виде круга со знаком вопроса внутри)
-	Кнопка «Ок» (после нажатия выводится булева функция в поле вывода, можно нажимать много раз и каждый раз будет новая функция) и «Очистить» (очищает поле ввода и вывода);
-	Кнопка «На главную страницу» (или «Назад», либо сделать в виде стрелочки);

Справка должна содержать следующую информацию:

-	Количество аргументов – целое положительное число.
-	Ограничение: число аргументов не может превышать 5.
