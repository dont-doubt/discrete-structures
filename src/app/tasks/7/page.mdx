# Задание 7

Система предлагает рандомный вектор булевой функции от n аргументов (можно сгенерировать с помощью solveTask1(n)), где n – тоже рандомное число.
Пользователь вводит КНФ. Система проверяет правильно ли нет пользователь ввёл КНФ.

На странице должны быть:

-	Поле с описанием задания «Введите КНФ булевой функции»;
-	Поле с вектором булевой функции;
-	Поле для ввода КНФ (можно вводить с клавиатуры, а можно дополнительно сделать кнопки для пользователя как на калькуляторе, содержащие символы «x, 1, 2, 3, 4, 5, -, *, v, (, )» и кнопки «     » (стирает один символ), «delete» (стирает всё));
-	Поле вывода (можно похвалить пользователя и вывести «Молодец» при написании правильной КНФ, при это выделить его ответ зеленым цветом, в случае, если пользователь ответил неправильно, можно выделить его ответ красным цветом и написать «Неправильно, попробуй еще раз»);
-	Справка (можно сделать в виде круга со знаком вопроса внутри); (?)
-	Кнопка «Ок» (при нажатии выводится результат) и «Новая функция» (очищает поле вывода и обновляет булеву функцию);
-	Кнопка «На главную страницу» (или «Назад», либо сделать в виде стрелочки);

Справка должна содержать следующую информацию:

-	Для обозначения переменных нужно использовать «x» и «порядковый номер», например, x1 – для первой переменной, x2 – для второй и так далее;
-	Для обозначения конъюнкции нужно использовать «*», дизъюнкции – «v», отрицания «-»;
-	Пробелы ставить не нужно.
-	Пример: (x1vx2)*(x2v-x3).

Функция solveTask7 в task7.ts принимает на вход вектор в виде массива из 0 и 1 и строку (ответ пользователя). На выход она даёт «true/false».
