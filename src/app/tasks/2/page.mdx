# Задание 2

Пользователь вводит вектор функции, 0 или 1, номер аргумента, программа выводит остаточную.
На странице должны быть:

-	Три поля для ввода («вектор функции», «0 или 1» - какая будет остаточная, «номер аргумента»);
-	Поле вывода;
-	Поле с описанием задания «Введите вектор булевой функции, 0 или 1 и номер аргумента, чтобы получить соответствующую остаточную»;
-	Справка (можно сделать в виде круга со знаком вопроса внутри); (?)
-	Кнопка «Ок» (после нажатия выводится остаточная в поле вывода, при повторном нажатии, если входные данные не менялись, остаточная должна оставаться та же) и «Очистить» (очищает поля ввода и вывода);
-	Кнопка «На главную страницу» (или «Назад», либо сделать в виде стрелочки);

Справка должна содержать следующую информацию:

-	Вектор не должен быть пустым.
-	Длина вектора должна быть степенью двойки.
-	Вектор может состоять только из 0 и 1.
-	0 или 1 – нулевая или единичная остаточная соответственно.
-	Номер аргумента – целое положительное число.
-	Номер указанного аргумента не должен превышать число аргументов булевой функции.
-	Ограничение: число аргументов не может превышать 5.
