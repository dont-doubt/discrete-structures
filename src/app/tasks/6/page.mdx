# Задание 6

Система предлагает рандомный вектор булевой функции от n аргументов (можно сгенерировать с помощью solveTask1(n)), где n – тоже рандомное число.
Пользователь вводит ДНФ. Система проверяет правильно ли нет пользователь ввёл ДНФ.

На странице должны быть:

-	Поле с описанием задания «Введите ДНФ булевой функции»;
-	Поле с вектором булевой функции;
-	Поле для ввода ДНФ (можно вводить с клавиатуры, а можно дополнительно сделать кнопки для пользователя как на калькуляторе, содержащие символы «x, 1, 2, 3, 4, 5, -, *, v, (, )» и кнопки «     » (стирает один символ), «delete» (стирает всё));
-	Поле вывода (можно похвалить пользователя и вывести «Молодец» при написании правильной ДНФ, при это выделить его ответ зеленым цветом, в случае, если пользователь ответил неправильно, можно выделить его ответ красным цветом и написать «Неправильно, попробуй еще раз»);
-	Справка (можно сделать в виде круга со знаком вопроса внутри); (?)
-	Кнопка «Ок» (при нажатии выводится результат) и «Новая функция» (очищает поле вывода и обновляет булеву функцию);
-	Кнопка «На главную страницу» (или «Назад», либо сделать в виде стрелочки);

Справка должна содержать следующую информацию:

-	Для обозначения переменных нужно использовать «x» и «порядковый номер», например, x1 – для первой переменной, x2 – для второй и так далее;
-	Для обозначения конъюнкции нужно использовать «*», дизъюнкции – «v», отрицания «-»;
-	Пробелы ставить не нужно.
-	Пример: (x1*x2)v(x2*-x3).

Функция solveTask6 в task6.ts принимает на вход вектор в виде массива из 0 и 1 и строку (ответ пользователя). На выход она даёт «true/false».
